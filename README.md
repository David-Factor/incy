## Incy 

An __incy wincy__ [parsing](https://lexi-lambda.github.io/blog/2019/11/05/parse-don-t-validate/) library for Typescript.

- Small (> 200 LOC) 
- Dependency Free
- Fast
- Customisable

Docs coming sooner or later...
